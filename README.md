# K-Means Clustering Project

## Overview
This project demonstrates the implementation of **K-Means Clustering**, an unsupervised machine learning algorithm used to group data into clusters based on similarity.

The notebook walks through the full data science workflow:
- Data loading
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Applying K-Means clustering
- Evaluating optimal clusters
- Visualizing results

---

## Features
- Clean and well-structured data pipeline
- Elbow Method for optimal K selection
- Cluster visualization using matplotlib
- Scalable approach for real-world datasets
